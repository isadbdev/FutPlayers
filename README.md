Projeto FutPlayers

Status do projeto: Em desenvolvimento

Tecnologias aplicadas:
- Java SE
- JDBC
- MySQL
- NetBeans
- (Futuro: JPA)

Time de desenvolvedores:
- Isabella Costa

Objetivo do software:
Gerenciar clubes, técnicos, competições, jogadores e confederações de futebol de forma integrada, permitindo consultas, cadastros e remoção de dados.

Funcionalidades do sistema:
1. Deve permitir cadastrar novos clubes com nome, competições disputadas, presidente, vice-presidente e diretor de futebol.
2. Deve permitir cadastrar novos jogadores com nome, idade, altura, peso, nacionalidade e clube.
3. Deve permitir cadastrar novos técnicos com nome, idade e clube.
4. Deve permitir cadastrar novas confederações com nome, país, continente e presidente.
5. Deve permitir cadastrar novas competições com nome, clubes participantes e confederações responsáveis.
6. Deve permitir remover cada dado que foi cadastrado, com exceção daqueles que são obrigatórios para a existência do item no sistema.
7. Deve listar todos os dados exigidos na busca.
8. Deve associar jogadores, técnicos e clubes corretamente por meio de chaves.
9. Deve associar clubes, confederações, competições e seus respectivos administradores corretamente por meio de chaves.
